Описание некоторых команд системы Linux.

Общие
  • adduser pi - создаём пользователя pi
  • cat - отображение файла на консоли
  • cd - смена текущей директории
  • chmod 555 scriptname - выдача прав на чтение/исполнение любому пользователю в системе
  • chmod +rx scriptname - выдача прав на чтение/исполнение любому пользователю в системе
  • chmod u+rx scriptname - выдача прав на чтение/исполнение только "владельцу" скрипта
  • clear - очистка экрана консоли
  • cp - копирование файла или директории
  • date - текущее время и дата; date +"%H:%M" - вывод времени в 24 часовом формате
  • exit - выход
  • htop - мониторинг процессов Linux
  • less - отображение файла на консоли постранично
  • ls - показывает файлы в текущей директории
  • man - система помощи по командам
  • mkdir - создание новой директории
  • mv - перемещение файла или директории
  • nano - редактирование файла
  • nmcli dev wifi - отображает wifi точки доступа
  • passwd - устанавливает или изменяет пароль пользователя
  • pstree - отображает дерево процессов
  • pwd - показывает текущюю директорию
  • rm - удаление файла или директории с ключом -r
  • top - отображает список наиболее активных процессов
  • touch - создание нового файла
  • tzselect - настройка таймзоны
  • useradd - добавляет учетную запись нового пользователя в систему и создает домашний каталог для данного пользователя
  • userdel - удаляет учетную запись пользователя из системы и удалит соответствующие файлы
  • usermod -aG sudo pi - разрешаем пользователю pi выполнять sudo
  • w - выводит информацию о системе, список пользователей, подключенных к системе и процессы, связанные с пользователями
  • wget - скачивание файла из интернета
  • who - выводит список пользователей, работающих в настоящий момент в системе
Настройка GIT
  • git config --global user.name "My Name" - Настройка пользователя
  • git config --global user.email myEmail@example.com - Настройка email
  • git config credential.helper store - Чтобы сохранялся пароль
  • git config --unset credential.helper - Обнулить настройки этой возможности можно командой
  • git config --list - Посмотреть настройки (Для выхода клавиша q)
  • git config --global core.editor "'C:/Program Files (x86)/Notepad++/notepad++.exe' -multiInst -notabbar -nosession -noPlugin" - Настройка внешнего редактора
  • git pull - Получить все изменения с Github
  • git status - Отобразит текущее состояние проекта (что изменилось но не сохранено в репозитории)
